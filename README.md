# Averon Partners — Website

Modern, premium website for **Averon Partners**, a business development firm helping SaaS, AI, automation, and technology companies expand in Europe and acquire B2B clients.

## Tech Stack

- **Next.js 15** (App Router, TypeScript)
- **Tailwind CSS 4**
- **Framer Motion** for animations
- **Lucide React** for icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Features

- Dark/light mode toggle
- Smooth scroll-triggered animations
- AI chat assistant widget
- Responsive design (mobile-first)
- SEO-friendly metadata
- Contact form with validation
